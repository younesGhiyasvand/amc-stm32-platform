# Automatic Modulation Classification (AMC)

Bachelor Project

## Overview

Design and implementation of an Automatic Modulation Classification (AMC) system using real-world signals generated and acquired by STM32 microcontrollers.

The system consists of:

- STM32-based transmitter
- STM32-based receiver
- Dataset generation
- Signal processing
- Feature extraction
- Machine learning classification

## Hardware

- STM32F407 (TX)
- STM32F407 (RX)

## Initial Modulations

- BPSK
- BFSK
- QPSK

## Software

- STM32CubeIDE
- Python
- NumPy
- SciPy
- Scikit-learn
- Matplotlib
