# Interface Board Design

Version: 1.0

Status: Approved

Last Updated: 2026-06-18

---

# Objective

Define the architecture and functionality of the custom interface board used in the AMC platform.

The interface board provides all analog front-end functions required for signal generation and signal acquisition. It acts as a bridge between the STM32F401CEU6 development boards and the analog transmission channel.

---

# Overview

The interface board contains:

- External Digital-to-Analog Converter (DAC)
- External Analog-to-Digital Converter (ADC)
- Signal conditioning circuits
- Reconstruction filter
- Anti-aliasing filter
- Analog input/output connectors
- Clock distribution circuitry
- Power filtering circuitry

The STM32F401CEU6 development boards communicate with the interface board through SPI and GPIO signals.

# Design Requirements

## Functional Requirements

- Generate analog signals from digital samples
- Acquire analog signals and convert them to digital samples
- Support AMC experiments
- Support configurable modulation schemes
- Provide stable and repeatable measurements

---

## Performance Requirements

- Target sampling rate: 500 kS/s
- Real-time operation
- Low-noise analog path
- SPI-based communication
- Compatibility with STM32F401CEU6

---

# Main Functional Blocks

## DAC Section

Purpose:

Convert digital samples generated by the transmitter STM32 into analog signals.

Functions:

- Digital-to-analog conversion
- Sample reconstruction
- Signal generation

Interface:

- SPI communication
- DAC chip select
- DAC update control

Connected Signals:

- DAC_SCK
- DAC_MOSI
- DAC_CS
- DAC_LDAC

---

## Reconstruction Filter

Purpose:

Remove high-frequency spectral images produced by the DAC.

Functions:

- Signal smoothing
- Noise reduction
- Analog signal conditioning

Recommended Topology:

- Second-order low-pass filter
- Active filter implementation

---

## Analog Transmission Interface

Purpose:

Provide access to the physical communication channel.

Functions:

- Signal transmission
- Experimental channel insertion
- External measurement access

Connector Type:

- SMA connector

Signals:

- TX_OUT
- GND

---

## Analog Reception Interface

Purpose:

Receive analog signals from the transmission channel.

Functions:

- Signal acquisition
- Analog signal routing

Connector Type:

- SMA connector

Signals:

- RX_IN
- GND

---

## Input Protection Circuit

Purpose:

Protect the ADC against excessive input voltages.

Functions:

- Overvoltage protection
- Current limiting

Recommended Components:

- Series resistor
- Protection diodes

---

## Anti-Aliasing Filter

Purpose:

Limit signal bandwidth before sampling.

Functions:

- Aliasing reduction
- Noise filtering

Recommended Topology:

- Second-order low-pass filter
- Active filter implementation

---

## ADC Section

Purpose:

Convert received analog signals into digital samples.

Functions:

- Signal sampling
- Analog-to-digital conversion

Interface:

- SPI communication
- Conversion synchronization

Connected Signals:

- ADC_SCK
- ADC_MISO
- ADC_CS
- ADC_DRDY

---

# Clock Distribution

Purpose:

Provide timing synchronization for sampling operations.

Functions:

- DAC update timing
- ADC synchronization
- System timing reference

Input Signal:

- SAMPLE_CLK

Source:

- STM32 TIM1_CH1

---

# Power System

## Input Power

Source:

- STM32 Development Board

Supply Voltage:

- 3.3V

---

## Power Filtering

Purpose:

Reduce digital switching noise entering analog circuits.

Functions:

- Analog supply filtering
- Noise suppression

Recommended Methods:

- Decoupling capacitors
- Ferrite beads
- Ground plane separation

# PCB Design Considerations

- Separate analog and digital ground regions
- Keep ADC and DAC traces short
- Place decoupling capacitors close to IC power pins
- Minimize noise coupling between analog and digital sections
- Use a continuous ground plane whenever possible
- Route SPI signals away from sensitive analog nodes

---

# Future Expansion

The interface board architecture allows future integration of:

- Higher-speed ADC devices
- Higher-speed DAC devices
- Variable gain amplifiers
- Additional filters
- Experimental channel models

These features are outside the scope of the current AMC platform implementation.

---

# Final Decision

The AMC platform will use a dedicated interface board containing external ADC and DAC devices, analog filtering stages, protection circuitry, and analog input/output interfaces.

The STM32F401CEU6 development boards will perform digital control, timing generation, communication, and data streaming, while all analog conversion functions will be handled by the interface board.
