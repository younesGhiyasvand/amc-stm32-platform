# Objective

Select the electronic components required for implementation of the AMC platform.

---

# Current Status

Component selection is currently under evaluation.

The final hardware architecture has been approved, including:

- STM32F401CEU6 development boards
- Custom interface board
- External ADC
- External DAC
- Analog front-end circuitry

Specific component models have not yet been finalized.

---

# Components Under Evaluation

The following component categories will be selected during the next design phase:

- Analog-to-Digital Converter (ADC)
- Digital-to-Analog Converter (DAC)
- Operational Amplifiers
- Anti-Aliasing Filters
- Reconstruction Filters
- Protection Circuits
- Connectors
- Power Supply Components
- Passive Components
